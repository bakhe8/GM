# دليل لقطات التوثيق

هذا المجلد مخصص للقطات الشاشة المرجعية المرتبطة بدليل المستخدم والدليل السريع.

الحالة الحالية:
- ملفات اللقطات نفسها غير مرفقة حاليًا داخل المستودع.
- هذا الملف أُعيدت إضافته لإبقاء روابط التوثيق الداخلية سليمة.
- عند تجهيز لقطات فعلية لاحقًا، يجب حفظها هنا بالأسماء التالية.

## ملاحظات الالتقاط

- تُلتقط الصور من نسخة تشغيل مستقرة، ويفضل من build منشور.
- تُستخدم بيانات غير حساسة أو بيانات تجريبية فقط.
- إذا تغيرت الواجهة لاحقًا، تُحدّث اللقطات مع الحفاظ على نفس الأسماء قدر الإمكان.
