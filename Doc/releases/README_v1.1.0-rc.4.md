# GuaranteeManager v1.1.0-rc.4

هذه رابع نسخة مرشحة لإطلاق `v1.1.0`. جاءت بعد `rc.3` كإصلاح مانع ومحدود الأثر لنتيجة عالية في تدقيق قواعد العمل، بدون إضافة ميزات جديدة.

> **مهم:** هذه نسخة Release Candidate. لا تضيف ميزات جديدة فوقها إلا إذا كانت إصلاحًا مانعًا ومحدود الأثر.

## حالة هذه النسخة

- رقم النسخة داخل التطبيق: `v1.1.0-rc.4`
- نوع الإصدار: **Release Candidate**
- الغرض: تثبيت خط `v1.1.0` بعد إغلاق فجوة التعديل الإداري في طبقة الخدمة
- اسم الحزمة: `GuaranteeManager_v1.1.0-rc.4_win-x64.zip`
- مجلد النشر: `releases/v1.1.0-rc.4`

## ما تغيّر بعد rc.3

- أضيف تحقق في `GuaranteeRepository.UpdateGuaranteeWithAttachments` يمنع تغيير الحقول التشغيلية عبر مسار التعديل الإداري حتى عند تجاوز الواجهة.
- أصبح تغيير رقم الضمان أو البنك أو النوع أو المبلغ أو تاريخ الانتهاء أو تقويم التاريخ أو بيانات المرجع أو الحالة التشغيلية أو روابط الاستبدال يرفع `OperationalFieldMutationException`.
- تم تعديل رسالة الواجهة لتتعامل مع تغيير تقويم التاريخ ضمن الحقول التشغيلية المحظورة.
- أضيف اختبار مباشر يثبت أن طبقة الخدمة ترفض تغيير الحقول التشغيلية وتحافظ على النسخة الحالية بدون إنشاء إصدار جديد.
- تم تحديث اختبارات staleness للتمديد والتخفيض حتى تستخدم مسار إنشاء نسخة تشغيلية بدل مسار التعديل الإداري المحمي.

## نطاق التجميد

- الواجهة الرسمية الحالية هي `اليوم`، `الضمانات`، `البنوك`، `التقارير`، و`الإعدادات`.
- لا توجد واجهة مستقلة للطلبات أو التنبيهات أو ملف الضمان.
- الطلبات وردود البنك والمرفقات تظهر وتدار من السجل الزمني، اللوحة الجانبية، اليوم، والتقارير.
- المستفيد ثابت: `مستشفى الملك فيصل التخصصي ومركز الأبحاث`.
- المورد هو الطرف المتغير الذي يظهر للمستخدم في الجداول، البحث، التقارير، ومخرجات Excel.

## قواعد العمل المثبتة

- طلب النقض ملغى تشغيليًا.
- الضمان المنتهي لا يقبل إلا الإفراج/الإعادة للبنك.
- التمديد والتخفيض يمكن أن يتواليا على الضمان النشط غير المنتهي.
- الإفراج والتسييل والاستبدال تنهي دورة حياة الضمان عند تأكيد البنك.
- تنفيذ إجراء نهائي يلغي الطلبات العالقة الأخرى على نفس الجذر.
- التعديل الإداري لا يغير المبلغ أو تاريخ الانتهاء أو تقويم التاريخ أو البنك أو رقم الضمان أو النوع أو المرجع أو الحالة التشغيلية.
- السجل الزمني هو مصدر الحقيقة لدورة حياة الضمان.
