# الدليل اليومي السريع

**Updated:** 2026-05-01

## أين أبدأ؟

- افتح `اليوم` لمعرفة الأولويات.
- افتح `الضمانات` عند العمل على ضمان محدد.
- افتح `التقارير` عند الحاجة إلى قراءة واسعة أو تصدير.

## ماذا أفعل حسب الحاجة؟

| الحاجة | المكان |
|---|---|
| إدخال ضمان جديد | `الضمانات` |
| فهم حالة ضمان واحد | `الضمانات` ثم السجل الزمني في اللوحة الجانبية |
| إنشاء طلب تمديد/تخفيض/إفراج/تسييل/تحقق/استبدال | أزرار الإجراءات في لوحة الضمان |
| تسجيل رد البنك | زر `تسجيل رد` أو حدث الطلب في السجل الزمني |
| إلحاق مستند رد متأخر | زر المرفق/الإلحاق أمام حدث الطلب |
| مراجعة المرفقات | أزرار الأحداث أو `عرض كل المرفقات` |
| تحليل عدة ضمانات | `التقارير` |

## قواعد سريعة

- المورد هو الاسم المهم المتغير في القوائم.
- الجهة المستفيدة ثابتة للنظام.
- لا يوجد طلب نقض.
- الضمان المنتهي لا يقبل إلا الإفراج/الإعادة للبنك.
- التمديد والتخفيض يمكن أن يتواليا ما دام الضمان نشطًا وغير منته.
- الإفراج والتسييل والاستبدال إجراءات نهائية بعد تأكيد البنك.

## قراءة السجل الزمني

اقرأ الأحداث بهذا الترتيب:

1. إنشاء الضمان.
2. أي إصدارات ناتجة عن تمديد أو تخفيض.
3. الطلبات البنكية.
4. ردود البنك.
5. المرفقات المرتبطة بكل حدث.
6. حالة النهاية إن وجدت: مفرج، مسيّل، أو مستبدل.
