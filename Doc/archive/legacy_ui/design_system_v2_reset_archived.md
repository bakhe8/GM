# Guarantee Manager - Design System V2 Reset

هذه الوثيقة كانت مرجعًا انتقاليًا لإعلان بداية `UI_V2`، ثم جرى أرشفتها حتى لا يبقى أي design system نشط في جذر المشروع.

## القرار المؤرشف

- الواجهة الحالية تعد **واجهة قديمة Legacy**
- لا تبنى الواجهة القادمة فوق `Views/`, `MainWindow.xaml`, `Themes/`, `ViewModels/` الحالية بوصفها مرجعًا تصميميًا
- أي عمل UI جديد يجب أن ينطلق من `UI_V2/`

## لماذا أُرشفت

لأن المطلوب لاحقًا كان أوضح:

- حتى ملف `design_system.md` الأرشيفي نفسه يجب ألا يبقى مرجعًا نشطًا
- المرجع النشط يجب أن يعيش بالكامل داخل `UI_V2/Specs`
