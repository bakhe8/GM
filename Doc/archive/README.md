# Documentation Archive

هذا المجلد يحتوي وثائق تاريخية أُخرجت من المراجع اليومية أثناء تنظيف 2026-05-03.

## الأقسام

- `releases/`
  - ملاحظات الإصدارات السابقة قبل مرشح الإطلاق الحالي.
- `ui-automation/`
  - أدلة أداة الواجهة ومخرجات القبول البصري القديمة.
- `presentation/`
  - ملفات README قديمة من داخل طبقة العرض.
- `assets/`
  - صور ومراجع بصرية قديمة لم تعد baseline نشطًا.

## قاعدة الاستخدام

لا تُحدّث ملفات الأرشيف إلا لتصحيح رابط مكسور يمنع قراءة التاريخ. أي توثيق حي يجب أن يكون في `Doc/README.md`.
